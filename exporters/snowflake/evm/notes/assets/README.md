Golden data for C chain

```sql
SELECT *
FROM AVALANCHE_PRIMARY_NETWORK_DATA.PRIMARY.[table name]
WHERE BLOCKNUMBER BETWEEN (75000000) AND (75100000)
  AND MOD(BLOCKNUMBER, 100) = 0
ORDER BY BLOCKNUMBER DESC;
```
